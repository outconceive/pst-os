# Desktop

PST OS boots to a windowed desktop rendered from Markout.

## Windows

The desktop has multiple windows. Each window has its own Markout document. Tab switches focus between them.

## Button Bar

GUI buttons at the bottom of the screen:

| Button | Key | Action |
|--------|-----|--------|
| Editor | F1 | Open text editor (.txt) |
| Markout | F2 | Open Markout editor (.md) |
| Browser | F3 | Open dt:// or gh:// browser |
| Code | F4 | Open code stepper |
| Form | F6 | Open login form demo |

Click buttons with the mouse or use F-keys.

## Mouse

- **Bullseye cursor** — white ring, blue iris
- **Hover** — iris turns green over clickable buttons
- **Click** — activates buttons, focuses input fields
- **Double-buffer compositing** — clean cursor rendering

## Keyboard

- **Tab** — switch windows
- **Esc** — save desktop state
- **F1-F6** — launch apps
- **Typing** — goes to focused window's input
